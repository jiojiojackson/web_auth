Digital Authentication Service

This service provides a simple digital authentication mechanism. The client displays a number, and the user must select the correct number from a set of three on the authentication page.

API Usage

POST /api/challenge

Creates a new authentication challenge.


    curl -X POST /api/challenge

Returns:


    {
      "challengeId": "...",
      "correctNumber": 2
    }

POST /api/verify

Verifies an authentication challenge.


    curl -X POST /api/verify -H "Content-Type: application/json" -d '{ "challengeId": "...", "selectedNumber": 2 }'

Returns:


    {
      "success": true
    }
